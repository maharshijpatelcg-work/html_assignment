HTML Entities & Symbols Reference

HTML entities are used to display reserved characters and special symbols that cannot be typed directly on a keyboard. This poster categorizes some of the most common ones.

Currency | Math | Arrows | Emoticons | Greek | Misc

Currency Symbols

  • ₹ — &#8377; : Indian Rupee
  • $ — &dollar; : Dollar
  • € — &euro; : Euro
  • £ — &pound; : Pound
  • ¥ — &yen; : Yen
  • ₩ — &#8361; : Won
  • ₽ — &#8381; : Russian Ruble
  • ₫ — &#8363; : Vietnamese Dong
  • ₴ — &#8372; : Hryvnia
  • ₦ — &#8358; : Naira

Mathematical Symbols

  • ± — &plusmn; : Plus-minus
  • × — &times; : Multiplication
  • ÷ — &divide; : Division
  • ≤ — &le; : Less than or equal
  • ≥ — &ge; : Greater than or equal
  • ≠ — &ne; : Not equal
  • ≈ — &asymp; : Approximately
  • ∞ — &infin; : Infinity
  • √ — &radic; : Square root
  • ∑ — &sum; : Summation
  • ∏ — &prod; : Product
  • ∫ — &int; : Integral
  • ∆ — &Delta; : Delta (difference)
  • ∂ — &part; : Partial derivative
  • ∇ — &nabla; : Nabla/gradient
  • ∴ — &there4; : Therefore
  • ∵ — &#8757; : Because
  • ° — &deg; : Degree
  • ∠ — &ang; : Angle
  • ≡ — &equiv; : Identical

Arrows

  • ← — &larr; : Left arrow
  • → — &rarr; : Right arrow
  • ↑ — &uarr; : Up arrow
  • ↓ — &darr; : Down arrow
  • ↔ — &harr; : Left-right arrow
  • ⇐ — &lArr; : Double left arrow
  • ⇒ — &rArr; : Double right arrow
  • ⇑ — &uArr; : Double up arrow
  • ⇓ — &dArr; : Double down arrow
  • ↕ — &#8597; : Up-down arrow
  • ↦ — &#8614; : Right arrow with hook
  • ↩ — &#8617; : Left arrow with hook
  • ↪ — &#8618; : Right arrow with hook
  • ⟵ — &#10229; : Long left arrow
  • ⟶ — &#10230; : Long right arrow
